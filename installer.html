UNIT3D Installer

Thinking about using UNIT3D? Here is how easy it is to install it.

Installer Preview

                    You will need a new server with a fresh install of Unbutu 16.04 - 18.04. You will then want to
                    SSH to the server and run the following commands.

                        
                            git clone https://github.com/ServNX/UNIT3D-INSTALLER.git installer
                            cd installer
                            sudo ./install.sh
                        

                    Follow the step in your terminal from there! Done!

                    For more info please visit 
                    https://github.com/ServNX/UNIT3D-INSTALLER
